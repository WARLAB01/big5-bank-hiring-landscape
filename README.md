# Big 5 Canadian Bank Hiring Landscape

A point-in-time analysis of public hiring data across Canada's Big 5 banks (BMO, RBC, TD, CIBC, Scotiabank), built as an interactive dashboard with Chart.js visualizations and a filterable data explorer.

**Live site:** [warlab01.github.io/big5-bank-hiring-landscape](https://warlab01.github.io/big5-bank-hiring-landscape/)

## Data Sources

- Indeed (via API connector) — listing-level detail
- LinkedIn (search results) — listing-level detail
- Bank career sites (BMO, RBC, Scotiabank) — aggregate counts
- Glassdoor/LinkedIn proxies (TD, CIBC) — estimated counts

## Methodology

Two-tier data approach:
- **Tier 1: Aggregate** — Career site totals (~5,289 roles across all 5 banks)
- **Tier 2: Sample** — 238 individual listings with full detail (title, location, date, source)

See the [Methodology section](https://warlab01.github.io/big5-bank-hiring-landscape/#methodology) on the live site for complete documentation.
